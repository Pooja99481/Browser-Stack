El País - Selenium Automation using BrowserStack

→ Objective:
Scrape top 5 article titles from: https://elpais.com/opinion/

→ Tech Used:
Python + Selenium + BrowserStack

→ Files:
- browser_test.py    → Main Execution
- config.py          → Browser Config
- credentials.py     → BrowserStack Credentials
- requirements.txt   → Python Libraries

→ Steps to Run:
1. pip install -r requirements.txt
2. Add credentials in credentials.py
3. python browser_test.py

→ Output:
- Browser Info
- Page Title
- First 5 Article Titles (or less if not available)
- BrowserStack Session Link
